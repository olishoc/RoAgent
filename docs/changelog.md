# RoAgent v3 — Changelog

## 2026-05-10 — Initial Build

- Modular architecture with 10 independent modules
- Custom script editor with full Lua syntax highlighting
- Inline suggestion popup with diff display and APPLY/DISMISS actions
- Suggestion queue with configurable max depth (default 5)
- Commit log panel with per-entry revert
- Per-script undo history (20 versions)
- Theme system with 4 built-in themes and custom theme support
- Agent bridge with connection state management and heartbeat
- Settings panel for server URL, polling interval, queue depth, script types
- Toolbar dropdown with connect, theme, settings, toggle
- Mock agent server for testing
- Unit test runner with 7 test files
- Documentation: architecture, ui, api, themes, testing, changelog
